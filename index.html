The Project

I wanted to make a ROM with a different look.
In 2018, I made OrangelOS, then I made WhileOS in 2019.
In 2020, I wanted an all new rom, with an all new look, but that is still familiar to AOSP.

AOSuiP was born. The main idea is to make an inexistant ROM, with only designs. Then after, we hope that we'll find developers for the project.
For the moment, we're making everything like it's a real ROM, but it's not.
If you're a developper and you're interested in the project, join our telegram group!
I'm currently searching for contributors ;)
